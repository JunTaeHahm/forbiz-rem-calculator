# REM Multiply Calculator

모바일 환경 작업 시 px값에 2를 곱해 rem으로 자동 변환 해줍니다.

![](demo.gif)

## 설치
- 프로젝트 클론 후 npm i
- npm run package 실행
- 루트폴더에 생긴 .visx 파일 수동 설치 후 사용

## 사용법

-   Windows: `Alt + R`
-   MacOS: `Cmd + R`
-   단축키 실행 후 px값 입력 후 엔터 (px를 제외한 숫자만 입력)
